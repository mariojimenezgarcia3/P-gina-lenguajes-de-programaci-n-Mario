Guía de introducción

Aprende a programar

Tutoriales básicos para dar tus primeros pasos en el mundo del código.

Py
Python
Simple y legible · Ideal para principiantes
1
Instala Python desde python.org y verifica con python --version
2
Crea un archivo llamado hola.py con cualquier editor de texto
3
Ejecuta el archivo con python hola.py en la terminal
# Mi primer programa en Python print("¡Hola, mundo!")

Salida en consola

¡Hola, mundo!
JS
JavaScript
El lenguaje de la web · Sin instalación
1
No necesitas instalar nada — abre tu navegador y pulsa F12
2
Ve a la pestaña Consola (Console) en las herramientas de desarrollo
3
Escribe el código y pulsa Enter para ejecutarlo al instante
// Mi primer programa en JavaScript console.log("¡Hola, mundo!");

Salida en consola

¡Hola, mundo!
C++
C++
Potencia y control · Alto rendimiento
1
Instala un compilador (GCC/MinGW) y verifica con g++ --version
2
Crea un archivo hola.cpp con el código de abajo
3
Compila con g++ hola.cpp -o hola y ejecuta con ./hola
#include <iostream> int main() { std::cout << "¡Hola, mundo!"; return 0; }

Salida en consola

¡Hola, mundo!
Jv
Java
Robusto y multiplataforma · Muy usado en empresa
1
Instala el JDK desde oracle.com y verifica con java --version
2
Crea HolaMundo.java — el nombre del archivo debe coincidir con la clase
3
Compila con javac HolaMundo.java y ejecuta con java HolaMundo
// Mi primer programa en Java public class HolaMundo { public static void main(String[] args) { System.out.println("¡Hola, mundo!"); } }

Salida en consola

¡Hola, mundo!
